Stock Manager

Project Overview: Stock Manager is a robust Java-based application designed to manage stock inventories and customer data through a clean, modern UI. It supports adding, updating, and removing stocks, maintaining customer info, and facilitating stock transactions (buy/sell). All operations are integrated with an Oracle database with proper authentication and data integrity.

Key Features

Stock Management

  • Add New Stock: Name, symbol, price, quantity.
  • Remove Stock: By unique symbol.
  • Update Stock: Price and quantity.
  • Retrieve Stock: View all stock records.

Customer Management

  • Add New Customer: With name and email.
  • Remove Customer: By customer ID.
  • Update Customer: Edit details.
  • Retrieve Customer: View all records.

Stock Transactions

  • Purchase Stocks: Quantity decreases; transaction logged for customer.
  • Sell Stocks: Adjusts quantity, records sale history.

Data Persistence

Database Integration: All records are stored in Oracle DB tables for stocks, customers, and transactions. Secure authentication is managed via a database-driven users table, enabling both customer and admin login and role-based access.
